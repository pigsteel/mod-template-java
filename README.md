# Personal 26.2 Mod Template

This is a multi-loader mod template for Fabric and Neoforge utilizing stonecutter. This is the Java-only version 
of mod-template-kotlin. This is a fork of murderspagurder's stonecutter template with code from cassiancc's fork of the template (tysm to both ^.^)
